Part B of task 2

1) What is the internet?

The internet is a global network of interconnected computers that communicate and share data and information using standardized protocols.



2) How does the internet work?

The internet transfers data in small packets between devices through routers and servers using the TCP/IP protocol.



3) What is a domain name?

A domain name is a human-readable or user-friendly address (like google.com) that identifies a website on the Internet.



4) DNS and how it works

DNS (Domain Name System) translates domain names into IP addresses so browsers can locate websites.



5) Browsers and how they work

Browsers are programs that request web pages from servers, interpret the code (HTML, CSS, JS), and display it visually.



6) What are markup languages?

Markup languages are languages like HTML or XML that use tags to structure and describe data on web pages.



7) What is front end development?

Front end development is the creation of the visual and interactive parts of a website that users directly see and interact with.



8) What is http?

HTTP (Hypertext Transfer Protocol) is the communication protocol used to transfer web data between browsers and servers.



9) What is a browser?

A browser is a software application (like Chrome) used to access and view websites on the internet.



10) How do browsers work?

Browsers work by sending requests to web servers, downloading website files, interpreting them, and rendering the content for the user.



11) How does the web work?

The web runs on the internet. The users request web pages via browsers, the servers send data, and the browsers render the results using web technologies.
